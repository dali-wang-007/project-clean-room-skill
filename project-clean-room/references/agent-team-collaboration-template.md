# {项目名或任务名} Agent团队协作文件

## 一、文件用途

本文件用于支持当前项目或大任务的 AI 协作净化和跨窗口接管。

这里的“项目”是广义概念，不一定是公司正式立项的业务项目，也可以是一个需要持续推进的大任务、专题、方案、PRD、知识库、汇报材料、研究事项或长期输出事项。

当用户输入“净化”时，智能体应先更新本文件，再继续当前窗口协作。

当用户输入“接管”时，智能体应先学习本文件，输出项目理解和未来协作事项，并让用户确认理解是否正确。

## 二、固定触发语

| 触发语 | 使用场景 | 智能体动作 |
|---|---|---|
| 净化 | 当前窗口变长、变慢、上下文压缩或信息混乱 | 更新本文件，并说明净化结果 |
| 接管 | 新窗口、新工具、新同事或新智能体接手项目 | 读取本文件，输出学习总结，并请求用户确认 |
